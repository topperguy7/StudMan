About

StudMan is a lightweight, browser-based Student Management System designed to simplify academic record handling.

It allows users to create, view, and delete student records seamlessly using intuitive CRUD operations.

All data is stored securely in the browser using localStorage, ensuring persistence even after page refresh without requiring a backend server.

The application focuses on simplicity, speed, and usability while demonstrating core front-end development concepts such as DOM manipulation, event handling, form validation, dynamic table rendering, and client-side storage.
