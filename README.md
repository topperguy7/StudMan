🎓 StudMan – Student Management System

StudMan is a lightweight, browser-based Student Management System built using HTML, CSS, and JavaScript.<br>
It allows users to Create, View, and Delete student records with persistent storage using localStorage — no backend required.
<hr>

🚀 Live Features
<ul>
  <li>Create Student Records (Name, Email, Course, SGPA)</li>
  <li>Delete Student Records</li>
  <li>Dynamic Table Rendering</li>
  <li>Persistent Data using localStorage</li>
  <li>Dark / Light Theme Toggle</li>
  <li>Responsive Design (Mobile Friendly)</li>
  <li>Smooth Scroll Navigation</li>
  <li>Dropdown Feature Menu</li>
  <li>Modal-based Form UI</li>
</ul>
